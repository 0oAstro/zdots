# Rosé Pine for Zsh

All natural pine, faux fur and a bit of soho vibes for the classy minimalist.

Sets up the Rosé Pine color palette for your Zsh environment, including:

- `LSCOLORS` — BSD `ls` colors tuned for Rosé Pine
- `FZF_DEFAULT_OPTS` — fzf interface themed to Rosé Pine
- Completion menu colors via `zstyle`
- Convenience color variables (`$ROSEPINE_BASE`, `$ROSEPINE_TEXT`, etc.) for use in prompts and scripts

## Install

### With Antidote

Add to `~/.config/zsh/.zsh_plugins.txt`:

```
yourusername/rose-pine-zsh kind:fpath path:script
```

### Manual

```zsh
source /path/to/rose-pine.plugin.zsh
```
